# Homebrew Tap for OxideTerm

Install OxideTerm with Homebrew:

```sh
brew install --cask AnalyseDeCircuit/tap/oxideterm
```

Or tap the repository first:

```sh
brew tap AnalyseDeCircuit/tap
brew install --cask oxideterm
```

OxideTerm is a local-first SSH workspace with terminals, SFTP, port forwarding,
file transfers, lightweight editing, and OxideSens AI context in one desktop app.

Project repository: <https://github.com/AnalyseDeCircuit/oxideterm>
